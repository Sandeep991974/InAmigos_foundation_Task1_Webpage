# 🌍 InAmigos Foundation – Frontend Website

A modern, responsive, and user-friendly frontend website developed for **InAmigos Foundation**. The website is designed to showcase the organization's mission, initiatives, impact, and community engagement through a clean and interactive user interface.

## ✨ Features

- 🎨 Modern and responsive design
- 📱 Mobile-friendly layout
- 🚀 Smooth scrolling and animations
- 🧭 Sticky navigation bar
- 💡 Interactive UI components
- 📊 Animated impact statistics
- 🖼️ Gallery section
- 👥 Team section
- ❓ FAQ accordion
- 📬 Contact form
- 🌙 Clean and accessible user experience

---

## 🏗️ Website Sections

- Home
- About Us

- Footer
